## Forensic Cyber Security course at IST

**Project:**

Goal: This project is a multi-stage digital forensics investigation focused on a QR-code–based phishing campaign used to harvest user credentials. It involves analyzing web artifacts, disk images, and network traffic to uncover hidden data, reconstruct event timelines, and attribute attacker activity. The objective is to identify evidence of coordinated cybercrime, including data exfiltration and potential financial fraud.
